Nodi Companion

Estendi TWIZA Moneypenny a telefoni, tablet e altri computer. I tuoi occhi e le tue orecchie nel mondo reale.

📱 Cosa Sono i Nodi

I nodi sono app companion che connettono i tuoi dispositivi fisici al Gateway di OpenClaw. Permettono all'AI di accedere a fotocamera, posizione GPS, notifiche, schermo e molto altro — trasformando il tuo telefono in un'estensione sensoriale dell'assistente.

🖥️ Dispositivi Supportati

📱
iOS
App companion per iPhone e iPad. Camera, foto, posizione, notifiche.
🤖
Android
App companion per dispositivi Android. Tutte le funzionalità sensore.
🍎
macOS
App nativa macOS. Screen recording, automazione, camera ed esecuzione remota.
💻
Node Host (CLI)
Qualsiasi macchina con Node.js. Esecuzione comandi remota e tool routing.

🎯 Capacità dei Nodi

  • 📸 Camera — Scatta foto dalla camera frontale o posteriore, registra clip video.
  • 🖼️ Foto — Accedi alle foto più recenti dal rullino del dispositivo.
  • 🖥️ Screen Recording — Registra lo schermo del dispositivo per monitoraggio o assistenza.
  • 📍 Posizione — GPS con precisione configurabile (approssimata, bilanciata, precisa).
  • 🔔 Notifiche — Leggi, apri o rispondi alle notifiche del dispositivo.
  • ⚡ Esecuzione Remota — Esegui comandi sulla macchina remota (con sistema di approvazione).
  • 📊 Info Dispositivo — Batteria, spazio disco, stato connessione, versione OS.

🔗 Collegamento (Pairing)

QR Code

Il modo più semplice. Apri l'app companion, seleziona "Connetti" e scansiona il QR mostrato dalla dashboard o dal terminale.

Setup Code

Per connessioni remote (VPS, Tailscale). Genera un codice dalla dashboard e inseriscilo nell'app companion.

Node Host CLI

Per collegare un'altra macchina come nodo di esecuzione:

openclaw node run --host gateway-ip --port 18789 --display-name "Build Server"
⚠️ Il pairing genera un token crittografico unico per dispositivo. I comandi eseguiti sui nodi richiedono approvazione esplicita dall'utente per sicurezza.

🔀 Tool Routing

L'AI può indirizzare i tool verso nodi specifici. Per esempio: "scatta una foto dalla camera del telefono" invia il comando al nodo mobile, mentre "esegui git pull sul build server" lo manda al Node Host configurato. Ogni nodo ha un nome e le sue capacità sono auto-rilevate.

Accesso Remoto: Se il Gateway è su loopback, puoi connettere nodi remoti tramite tunnel SSH: ssh -L 18789:localhost:18789 user@gateway-host
