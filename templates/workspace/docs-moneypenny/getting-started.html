Guida Introduttiva

Inizia subito con TWIZA Moneypenny

TWIZA Moneypenny è la versione preconfigurata di OpenClaw con tutte le impostazioni ottimizzate per il massimo delle performance.

1

Installa Node.js

TWIZA Moneypenny richiede Node.js versione 24 o superiore (minimo 22.14+). Scarica l'ultima versione LTS dal sito ufficiale.

# Verifica la versione installata
node --version
⚠️ Importante: Assicurati di avere almeno Node.js v22.14+ per evitare problemi di compatibilità.
2

Installa OpenClaw

Installa OpenClaw usando il nostro script di installazione automatica. Funziona su macOS, Linux e Windows.

macOS e Linux:

curl -fsSL https://openclaw.ai/install.sh | bash

Windows (PowerShell come Amministratore):

iex ((New-Object System.Net.WebClient).DownloadString('https://openclaw.ai/install.ps1'))
💡 Suggerimento: L'installazione richiede alcuni minuti. Al termine, riapri il terminale per aggiornare il PATH.
3

Configurazione Iniziale

Esegui il processo di onboarding per configurare TWIZA Moneypenny con tutte le impostazioni ottimali.

openclaw onboard --install-daemon

Questo comando:

  • Configura il daemon del Gateway
  • Imposta i modelli AI predefiniti
  • Configura i canali di comunicazione
  • Ottimizza le performance del sistema
4

Verifica l'Installazione

Controlla che il Gateway sia attivo e funzionante.

openclaw gateway status
✅ Output atteso: Gateway running on http://localhost:18789

Se il Gateway non è attivo, avvialo manualmente:

openclaw gateway start
5

Accedi al Dashboard

Apri il dashboard web di TWIZA Moneypenny per iniziare a configurare i tuoi canali e impostazioni.

openclaw dashboard

Questo aprirà automaticamente il browser su http://localhost:18789

💡 Suggerimento: Aggiungi il dashboard ai preferiti per accesso rapido.
6

Primo Messaggio

Invia il tuo primo messaggio nella webchat per testare che tutto funzioni correttamente.

Nel dashboard, clicca su "WebChat" nella barra laterale e prova a inviare:

Ciao Moneypenny! Come stai?
✅ Congratulazioni! TWIZA Moneypenny è ora attivo e pronto all'uso!

Prossimi Passi:

  • Configura i tuoi canali preferiti (WhatsApp, Telegram, Discord...)
  • Personalizza i modelli AI e le impostazioni
  • Esplora le skill e i plugin disponibili
  • Leggi la documentazione completa per funzionalità avanzate
