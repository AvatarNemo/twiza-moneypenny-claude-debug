# TWIZA Moneypenny — Style Guide

*...more than an agent!*

Based on SHAKAZAMBA/TWIZA branding from shakazamba.com and official assets.

## Color Palette

### Primary Colors
- **TWIZA Fuchsia/Magenta**: `#D4006A` / `#D30F63` — primary accent, CTAs, highlights
- **TWIZA Purple**: `#6600FF` — secondary accent, links, hover states
- **SHAKAZAMBA Green**: `#5EC900` / `#61CE70` — success states, confirmations

### Background Colors  
- **Deep Black**: `#010205` — main background
- **Dark Navy**: `#050913` — card backgrounds, panels
- **Dark Gray**: `#0D111A` — secondary panels
- **Medium Dark**: `#171420` — elevated surfaces
- **Card Gray**: `#2A2E36` — input fields, cards

### Text Colors
- **White**: `#FFFFFF` — primary text
- **Light Gray**: `#74727B` — secondary text, labels
- **Muted**: `#54595F` — disabled, placeholder text

### Accent Colors
- **Orange Warm**: `#FFBC7D` — warnings, badges
- **Cyan/Teal**: `#6EC1E4` — info, links alternative
- **Dark Purple**: `#370039` / `#440c2a` — gradients

## Gradients
- **Primary gradient**: `linear-gradient(135deg, #D4006A, #6600FF)` — buttons, headers
- **Background gradient**: `linear-gradient(180deg, #010205, #050913, #0D111A)` — page background
- **Accent gradient**: `linear-gradient(135deg, #440c2a, #370039)` — cards hover

## Typography
- **Headings**: Clean sans-serif, bold, white
- **Body**: Sans-serif, regular, white/light gray
- **Suggested fonts**: Inter, Outfit, or system-ui
- **Sizes**: 
  - H1: 2.5rem
  - H2: 1.8rem
  - Body: 1rem
  - Small: 0.85rem

## UI Components

### Buttons
- **Primary**: Gradient background (#D4006A → #6600FF), white text, rounded (8px)
- **Secondary**: Transparent, border 1px #6600FF, purple text
- **Hover**: Brightness increase, subtle glow

### Input Fields
- Background: `#2A2E36`
- Border: 1px solid `#54595F`
- Focus border: `#D4006A`
- Text: white
- Placeholder: `#74727B`
- Border radius: 8px

### Cards/Panels
- Background: `#050913` or `#0D111A`
- Border: 1px solid rgba(255,255,255,0.05)
- Border radius: 12px
- Subtle shadow

### Wizard Steps
- Active step: Fuchsia indicator
- Completed: Green checkmark
- Upcoming: Gray/muted

## Logo Usage
- **Dark backgrounds**: Use `twiza-white.png` or `twiza.svg`
- **Light backgrounds**: Use `twiza-black.png`
- **Icon only**: Use `twiza-icon.png`
- **Full logo + text**: Use `twiza-logo-text.png`

## Brand Tone
- Professional but approachable
- Tech-forward, innovative
- Ethical AI, privacy-first
- European identity (Italian roots, EU compliance)
- Clean, modern, slightly futuristic
