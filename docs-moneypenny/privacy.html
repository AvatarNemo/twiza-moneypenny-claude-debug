Privacy e Sicurezza

🔒 La tua privacy è sacra

TWIZA Moneypenny è progettata con la privacy al centro. I tuoi dati restano tuoi, sempre.

🛡️ I Nostri Principi

🏠

Local-First

Tutto funziona sul tuo computer. Nessun server esterno necessario per l'uso base.

🎛️

Tu Controlli

Decidi tu se e quando condividere dati con servizi esterni. Ogni scelta è nelle tue mani.

🔍

Trasparente

Ti diciamo sempre esattamente cosa facciamo con i tuoi dati. Niente scatole nere.

🚫

Zero Tracking

Non raccogliamo telemetria, non abbiamo analytics, non tracciamo il tuo utilizzo.

📊 I Tuoi Dati

🔒 Sempre Locali

  • Le nostre conversazioni
  • I miei ricordi su di te (MEMORY.md)
  • I file che elaboro
  • Configurazioni e impostazioni
  • Cronologia comandi
  • I tuoi progetti e lavori

Questi dati non lasciano mai il tuo computer a meno che tu non lo richieda esplicitamente.

☁️ Solo se Scegli Cloud

  • Solo il messaggio corrente (per ottenere risposta)
  • Contesto essenziale quando necessario

Solo quando usi modelli cloud (Anthropic, OpenAI, ecc.). Il provider riceve solo quello che serve per rispondere.

🚫 Mai Condiviso

  • Le tue API key (restano sul tuo PC)
  • Password e credenziali
  • File personali senza permesso
  • Log e diagnostici completi
  • Metadata delle tue attività

Questi dati non escono mai dal tuo computer. Punto.

☁️ Privacy con Modelli Cloud

Quando scegli di usare modelli cloud (opzionale), ecco cosa succede:

🧠 Anthropic Claude

✅ Non usa i tuoi dati per training
✅ Non conserva conversazioni
✅ Conformità GDPR/Privacy

Politica privacy molto rispettosa. Dati processati solo per rispondere.

🦾 OpenAI GPT

⚠️ Potrebbe usare dati per miglioramenti*
✅ Puoi richiedere cancellazione
✅ API più private della chat web

*Puoi disattivare il training nelle impostazioni del tuo account OpenAI.

🌟 Google Gemini

⚠️ Integrato con ecosistema Google
✅ Controlli privacy granulari
✅ Report trasparenza disponibili

Rispetta le tue impostazioni Google esistenti.

🛡️ Sicurezza Tecnica

💾 Storage Locale

  • Dati crittografati nel tuo profilo utente
  • API key protette da Windows DPAPI
  • Accesso solo al tuo account Windows

🌐 Sicurezza di Rete

  • Comunicazioni sempre HTTPS/TLS
  • Certificati SSL verificati
  • Nessun proxy o man-in-the-middle

🏗️ Isolamento Processi

  • Sandbox WSL2 per AI locale
  • Electron con security abilitata
  • Permessi minimi necessari

❓ Domande Frequenti

Posso usare TWIZA Moneypenny completamente offline?

Sì! Il modello locale funziona senza internet. Serve connessione solo per browsing web e modelli cloud opzionali.

Le mie API key sono al sicuro?

Assolutamente. Sono crittografate localmente con Windows DPAPI. Solo il tuo account può accedervi.

Cosa vede il provider cloud?

Solo il messaggio che invii e il contesto minimo necessario. Mai file, cronologia completa o dati personali.

Posso eliminare tutto in qualsiasi momento?

Certo. Tutti i dati sono nella cartella utente. Puoi eliminarli o fare reset completo quando vuoi.

C'è telemetria nascosta?

Zero. Niente analytics, niente crash reports automatici, niente tracking. Il tuo utilizzo resta privato.

E se SHAKAZAMBA chiude?

I tuoi dati restano tuoi. OpenClaw è open source, puoi continuare a usarlo indipendentemente.

🤝 Il Nostro Impegno

TWIZA Moneypenny è costruita per essere tua. Non siamo un servizio che controlli — sei tu che controlli il servizio. La tua privacy non è un prodotto da vendere, ma un diritto da rispettare.

"Non raccoglieremo mai i tuoi dati per profitto. Non ti tracceremo mai. Non venderemo mai niente che ti riguardi. Punto."

— Il Team SHAKAZAMBA
